Project: Mobile Media

social sports connection in a foreign country

The goal of this project was to create an app that facilitates social interaction through sports, which aids in a person's integration into a new nation. The software was created to help new users locate local sports communities and form friendships, using the example of a women's football club in New Zealand. I worked on the following during the project:
✔ Competitive analysis to evaluate current solutions.
✔ Coming up with ideas and defining the app's main features.
✔ Outlining the user journey with storyboards, flowcharts, and user scenarios.
✔ Creating a structured menu navigation and wireframes.
✔ Creating mock-ups and a style guide to ensure a unified visual identity.
✔ Creating a clickable Figma prototype to show off the features of the app.
This methodical process produced an app design that is easy to use and promotes social interactions.
